The text-overflow Property

The following two divs contains a text that will not fit in the box.

text-overflow: clip (default):

Hello world!

text-overflow: ellipsis:

Hello world!

text-overflow: "----" (user defined string):

Hello world!

Overflow: Auto, visible, hidden, clip, scroll

The text-overflow property specifies how overflowed content that is not displayed should be signaled to the user. It can be clipped, display an ellipsis (...), or display a custom string BBB.
